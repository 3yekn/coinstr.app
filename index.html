coinstr.app  
 
     Coinstr is an app that uses Bitcoin and Nostr to orchestrate multi-custody taproot/schnorr signature requests.

     1. Create complex UTXO spending policies using no code building blocks.     
     2. Signature requests will be generated and orchestrated via nostr.
     3. Users sign requests, merged, and broadcast
